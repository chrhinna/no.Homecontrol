# Homecontrol Zigbee

This app allows you to connect your Homecontrol devices to Homey

# Supported Devices

* HC GU10 bulb
* HC E27 bulb
* Develco develco zhwr202

# Devices in progress

* HCO (Dual plug)
* HC Motion sensor
* HC Door/Window sensor
